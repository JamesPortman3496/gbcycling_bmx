# BMX Freestyle Park Performance Tracker

Interview task prototype for tracking BMX Freestyle Park performance.

## Getting started

```sh
make install
make dev
```

The app runs at http://localhost:3000.

## Available commands

```sh
make check
make prepare-data
```

The Makefile runs commands with Node.js 22. It uses the active Node version if
it already matches, otherwise it tries `nvm`, then Homebrew `node@22`.
